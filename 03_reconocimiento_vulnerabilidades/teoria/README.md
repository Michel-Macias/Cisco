## 🎯 Temas Críticos

- **Footprinting:** Recopilación de información sobre el dominio, red y personal.
- **Scanning:** Identificación de puertos abiertos y versiones de servicios.
- **Enumeration:** Extracción de nombres de usuario, carpetas compartidas y banners.
- **Vulnerability Assessment:** Comparativa contra bases de datos CVE/CWE.
