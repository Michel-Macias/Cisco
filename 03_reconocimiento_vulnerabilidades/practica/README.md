# 🛠️ Práctica - Módulo 03: Reconocimiento y Análisis de Vulnerabilidades

> Laboratorios prácticos organizados en 3 fases progresivas del ciclo de reconocimiento.

---

## 📂 Estructura

| # | Sección | Descripción | Labs |
|:--|:---|:---|:---|
| 01 | [Reconocimiento Pasivo](./01_reconocimiento_pasivo/) | Google Dorking, DNS, OSINT, SSL, Recon-ng | 7 labs |
| 02 | [Reconocimiento Activo](./02_reconocimiento_activo/) | Scapy, Nmap, Wireshark + memorias técnicas | 3 labs + 3 memorias |
| 03 | [Análisis de Vulnerabilidades](./03_analisis_vulnerabilidades/) | Metodología de escaneo, CVE/CVSS/CWE | 2 labs |

---

## 🗺️ Flujo Recomendado

```
Reconocimiento Pasivo (sin tocar al objetivo)
  → Google Dorking → DNS/Whois → OSINT → SSL → Recon-ng
       ↓
Reconocimiento Activo (interacción directa)
  → Scapy (paquetes custom) → Nmap (escaneo) → Wireshark (captura)
       ↓
Análisis de Vulnerabilidades (evaluación)
  → Metodología de escaneo → Investigación CVE/CVSS
```
